# 第1章 §1 整式の計算（Polynomial Operations）

## 1. 同類項の整理（Combine Like Terms）

### 基本公式
(a₁xⁿ + a₂xⁿ) = (a₁ + a₂)xⁿ  
同じ文字・同じ指数をもつ項をまとめる。

**例題**
4x² − 2x² + 3x − x + 3 − 5  
= (4−2)x² + (3−1)x + (3−5)  
= 2x² + 2x − 2  

**ポイント**
- 符号を正確に処理する。  
- 項の順番をそろえると見やすい。

---

## 2. 多項式の加減（Addition and Subtraction of Polynomials）

### 基本公式
(A ± B) = 各項を同類項ごとに加減  

**例題**
A = 2x² + 3x + 1  
B = 3x² − 6x + 2  
A + B = 5x² − 3x + 3  
A − B = −x² + 9x − 1  

**ポイント**
- 括弧の符号変化を見落とさない。

---

## 3. 乗法公式（Multiplication Formula）

| 形式 | 展開 |
|------|------|
| (a + b)² | a² + 2ab + b² |
| (a − b)² | a² − 2ab + b² |
| (a + b)(a − b) | a² − b² |
| (a + b)³ | a³ + 3a²b + 3ab² + b³ |
| (a − b)³ | a³ − 3a²b + 3ab² − b³ |

**例題**
(3a + b)³ = 27a³ + 27a²b + 9ab² + b³  
(2a − 3b)³ = 8a³ − 36a²b + 54ab² − 27b³  

---

## 4. 因数分解の公式（Factorization Formulas）

| 形式 | 因数分解 |
|------|----------|
| a² − b² | (a + b)(a − b) |
| a³ + b³ | (a + b)(a² − ab + b²) |
| a³ − b³ | (a − b)(a² + ab + b²) |
| ax + ay | a(x + y) |
| ax + bx | (a + b)x |

**例題**
3(x² − 9y²) = 3(x + 3y)(x − 3y)  
(a² + 6ab + 9b²) − 4c² = (a + 3b + 2c)(a + 3b − 2c)  

---

## 5. 二次式の因数分解（Quadratic Polynomials）

### 標準形
x² + px + q = (x + α)(x + β)  
ただし α + β = p, αβ = q  

**例題**
x² + 7x + 10 = (x + 2)(x + 5)  
x² + (6−1)x + (−1)×6 = (x + 6)(x − 1)  

---

## 6. 剰余の定理（Remainder Theorem）

f(x) を (x − a) で割った余りは f(a)。

**例題**
f(x)=x³−3x²−x+4  
f(1)=1−3−1+4=1  
→ 余り=1  

---

## 7. 割り切れる条件（Divisibility Condition）

f(x) が (x − a) で割り切れる ⇔ f(a)=0  

**例題**
P(x)=x³+2x−12  
P(2)=8+4−12=0 → x−2で割り切れる。  

---

## 8. 未知係数の決定（Coefficient Determination）

f(x)=(x³−3x²+kx−4) が x−2で割り切れるとき  
f(2)=0 → 8−12+2k−4=0 → 2k=8 → k=4  

---

## 9. 最大公約数・最小公倍数（GCD and LCM of Monomials）

| 操作 | 方法 |
|------|------|
| GCD | 各文字の指数の小さい方 |
| LCM | 各文字の指数の大きい方 |

**例題**
a b² と b c → GCD=b, LCM=a b² c  

---

## 10. 整式の除法（Polynomial Division）

A = BQ + R   （余りRの次数 < Bの次数）

**例題**
A ÷ B = 2x + 1, 余り2  
→ A = B(2x + 1) + 2  

---

## 11. 剰余の一般化（余りの定数化）

f(x) ÷ (2x − 1)  
→ 余り = f(½)  

**例題**
f(x)=4x³+2x²−3x+2  
f(½)=3/2 → 余り 3/2  

---

## 12. 高次因数分解（High-order Factorization）

**例題**
x⁴ + 3x³ − 2x² − 12x − 8  
→ f(−1)=0 ⇒ (x + 1)で割れる  
→ 商 = x³ + 2x² − 4x − 8  
→ f(2)=0 ⇒ (x − 2)で割れる  
→ f(x)=(x + 1)(x − 2)(x + 2)²

# 第1章 §2 いろいろな数と式（Various Numbers and Expressions）

## 1. 分数式の計算（Fractional Expressions）

(A/B) ÷ (C/D) = (A/B) × (D/C)  
(AC)/(BD) = (A/B)×(C/D)  

**例題**
(3³x³y⁶)/(3²x⁴y²z⁶) = (3y⁴)/(xz⁶)  

---

## 2. 分数式の加減（Addition/Subtraction of Fractions）

1. 通分して分母をそろえる  
2. 分子を整理する  
3. 因数分解で簡約する  

**例題**
(1(x−1))/((x+1)(x−1)) + (2(x+1))/((x−1)(x+1))  
= (3x+1)/((x+1)(x−1))  

**注意**
- 分母に0を与える値を除外。  
- 共通因数があれば約分。

---

## 3. 分数式の乗除（Multiplication and Division）

(3b²c×4a³)/(8a×9bc) = (a²b)/6  

---

## 4. 分数式の変形（Simplification）

(1 − 1/x) × x / ((x − 1/x)×x) = 1/(x+1)  

---

## 5. 整式の除法（Polynomial Division）

A ÷ B = Q + (R/B)  
例：3x−2 + 3/(x+2)  

---

## 6. 絶対値（Absolute Value）

|a| =  
a （a ≥ 0）  
−a （a < 0）  

**例題**
|0−1|+|0−3|=4  

---

## 7. 根号を含む式の計算（Surds）

| 形式 | 結果 |
|------|------|
| √a√b | √(ab) |
| (√a)² | a |
| (a√b)(c√d) | ac√(bd) |

**例題**
2√3−3√3+4√3=3√3  

---

## 8. 絶対値と平方根の関係

√{(a−b)²} = |a−b|  

**例題**
√{(π−5)²} = |π−5| = 5−π  

---

## 9. 有理化（Rationalization）

| 形式 | 処理法 |
|------|--------|
| 1/√a | 分母に√aをかける |
| 1/(√a+√b) | 共役式(√a−√b)をかける |

**例題**
√2(√5+√3)/((√5−√3)(√5+√3)) = (√10+√6)/2  
(2√3−3)(2√3−3)/((2√3+3)(2√3−3)) = 7−4√3  

---

## 10. 虚数単位 i（Imaginary Unit）

i² = −1  

**例題**
(1+2i)(3+4i) = (−1/5) + (2/5)i  

---

## 11. 複素数の共役（Conjugate）

z = a + bi のとき、共役 z̅ = a − bi  

**例題**
3−2i の共役 → 3+2i  
|3+i| = √(3²+1²) = √10  

---

## 12. 絶対値の幾何的意味

| 形式 | 意味 |
|------|------|
| a | 原点からaまでの距離 |
| a+bi | 複素平面上の距離 √(a²+b²) |
